Configuration

在 Jounal 主题下，一个完整的 markdown 表头形如：

---
title: <title>
date: 1970-1-1 00:00:00
intro: <intro> # or use "<intro>" for spaced sentences
featured_image: img/69941255.png # or use `https://...` instead
category:
- <cat1>
- <cat2>
tags: 
- <tag1>
- <tag2>
toc: true  # to use Table of Contents
math: true # to use Mathjax Render for math formula
---

Inline Equations
